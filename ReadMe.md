# Hello GimpFu !!

　pythonによるGIMP自動化スクリプトの最小限コード。

# 使い方

```bash
git clone https://github.com/ytyaru/Python.Gimp.HelloWorld.20191223132647
cd Python.Gimp.HelloWorld.20191223132647/src
./run.sh
```
